🏦 CashNova – ATM Simulator 

CashNova is a console-based ATM Simulator developed using Object-Oriented Programming (OOP) principles in Java. The application simulates core banking operations such as authentication, balance inquiry, deposit, and withdrawal through a secure and menu-driven interface.

This project demonstrates strong understanding of Java fundamentals, encapsulation, and modular program design.

🚀 Features

🔐 Secure login using Username & PIN

💰 Balance inquiry

➕ Deposit functionality

➖ Withdraw functionality

⚠️ Input validation and error handling

📋 Menu-driven interactive system

🛠 Tech Stack

Java

OOP Concepts (Encapsulation, Abstraction, Constructors)

Scanner Class for user input

📂 Project Structure
CashNova.java


Classes Included:

Account – Handles user data and transaction logic

ATM – Manages user interaction and menu system

CashNova – Main driver class

▶️ How to Run

Compile the program:

javac CashNova.java


Run the program:

java CashNova

🎯 Objective

To simulate real-world ATM functionality while applying core Object-Oriented Programming principles and structured software design.
